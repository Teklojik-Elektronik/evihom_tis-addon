# TIS Laravel Addon (Free Version)

Laravel-based TIS Control addon for Home Assistant - **Backpack PRO removed, uses free CRUD only**.

## Changes from Original
- ❌ Removed Backpack PRO dependency ($299 license)  
- ✅ Uses free Backpack CRUD (sufficient for all features)
- ✅ All functionality preserved
- ✅ No license fees required

## Backpack PRO Note
Original addon required Backpack PRO but only used **BulkDeleteOperation** which is available in the free version. This version removes the unnecessary paid dependency.

## Features
- Device management via web UI
- Appliance configuration
- Virtual device support
- MQTT integration
- Home Assistant energy/lock integration
- **No paid licenses required**
